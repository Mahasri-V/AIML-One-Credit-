# My Portfolio

A modern, unique, and visually impressive personal portfolio website built with React.

## Features

- Responsive design
- Creative UI/UX with glassmorphism effects
- Smooth animations and transitions
- Functional components only

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

To build the project for production:

```bash
npm run build
```

## Technologies Used

- React
- Vite
- CSS3
